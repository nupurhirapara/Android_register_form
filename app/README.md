# Registration Form App

A simple and clean Android application that allows users to register their details.

## ✨ Features
* **User Input:** Fields for Name, Email, and Password.
* **Validation:** Checks for empty fields and valid email formats.
* **Navigation:** Smooth transition between Register and Login screens.
